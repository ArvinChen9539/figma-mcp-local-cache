# PowerPoint Skills

This skill provides capabilities for working with PowerPoint files, primarily using the `python-pptx` library.

## Capabilities

1.  **Create Presentations**: Generate new .pptx files from scratch.
2.  **Edit Slides**: Add slides, text, images, charts, and tables to existing presentations.
3.  **Extract Content**: Read text and metadata from slides.
4.  **Layout Management**: Use different slide layouts (Title, Bullet points, Blank, etc.).

## Prerequisites

Ensure `python-pptx` is installed in the environment:
```bash
pip install python-pptx
```

## Usage Examples

### 1. Create a Simple Presentation

```python
from pptx import Presentation

def create_presentation(filename="output.pptx"):
    prs = Presentation()
    
    # Slide 1: Title Slide
    slide_layout = prs.slide_layouts[0] # 0 is title slide
    slide = prs.slides.add_slide(slide_layout)
    title = slide.shapes.title
    subtitle = slide.placeholders[1]
    title.text = "Hello, World!"
    subtitle.text = "Generated by ppt-skills"

    # Slide 2: Bullet Points
    bullet_slide_layout = prs.slide_layouts[1]
    slide = prs.slides.add_slide(bullet_slide_layout)
    shapes = slide.shapes
    title_shape = shapes.title
    body_shape = shapes.placeholders[1]

    title_shape.text = "Key Points"
    tf = body_shape.text_frame
    tf.text = "Point 1"
    
    p = tf.add_paragraph()
    p.text = "Point 2"
    p.level = 1

    prs.save(filename)
    print(f"Saved {filename}")
```

### 2. Read Text from Presentation

```python
from pptx import Presentation

def read_presentation(filename):
    prs = Presentation(filename)
    text_runs = []
    for slide in prs.slides:
        for shape in slide.shapes:
            if hasattr(shape, "text"):
                text_runs.append(shape.text)
    return "\n".join(text_runs)
```

## Best Practices

- Always check if the file exists before trying to open it.
- Use `slide_layouts` to maintain consistent styling.
- Handle exceptions when saving files (e.g., file open in another program).
